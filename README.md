# 📱 Site Responsivo: Estudo de Layout Adaptável

> Projeto focado na criação de uma interface moderna que se adapta perfeitamente a qualquer tamanho de tela (Desktop, Tablet e Mobile).

Este repositório faz parte do meu portfólio de **Desenvolvimento Web**, onde aplico técnicas avançadas de CSS para garantir que a experiência do usuário seja consistente e fluida, independentemente do dispositivo utilizado.

---

## 🎨 O Projeto

O objetivo principal foi construir uma página do zero utilizando boas práticas de semântica e estilização moderna, focando em:
* **Mobile-First Design:** Priorização da experiência em dispositivos móveis.
* **Flexibilidade:** Uso de unidades relativas e layouts fluídos.
* **Performance:** Código limpo e otimização de ativos (imagens/fontes).

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica para melhor acessibilidade e SEO.
* **CSS3:**
    * **Flexbox & Grid Layout:** Para alinhamento e posicionamento dinâmico.
    * **Media Queries:** Para criação de breakpoints estratégicos.
    * **Variáveis CSS:** Para facilitar a manutenção de cores e fontes.
* **JavaScript (Opcional):** Para interações simples como menus hambúrguer.

---

## 🚀 Principais Aprendizados

Durante o desenvolvimento deste projeto, aprofunduei meus conhecimentos em:
1.  **Breakpoints:** Identificação dos pontos de ruptura ideais para cada layout.
2.  **Viewports:** Ajuste de meta tags para controle de escala em dispositivos móveis.
3.  **Arquitetura CSS:** Organização de estilos para evitar repetição de código (DRY - Don't Repeat Yourself).
